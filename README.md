# 🎓 SkillForge Academy

A responsive single-page website for a Career Training Institute built using HTML, CSS, JavaScript, and Node.js. This project was developed as part of a Website Developer Internship assignment.

---

## 🚀 Live Features

- Responsive design for desktop, tablet, and mobile
- Modern navigation bar with hamburger menu
- Hero section
- About section
- Popular Courses section
- Why Choose Us section
- Student Testimonials
- FAQ Accordion
- Contact Form
- Google Maps Integration
- Backend API using Node.js & Express
- Stores enquiry data in JSON
- Smooth scrolling animations
- Scroll reveal effects

---

## 🛠 Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript (ES6)

### Backend
- Node.js
- Express.js
- CORS
- Dotenv

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/ashokrajsakthivel/skillforge-academy.git
```

Go to project

```bash
cd SkillForge-Academy
```

Install backend packages

```bash
cd backend
npm install
```

Start Server

```bash
node server.js
```

Server runs at

```
http://localhost:3000
```

Open frontend

```
index.html
```

or use VS Code Live Server.

## 📬 Contact Form API

### POST

